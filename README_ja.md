# AI活用ポートフォリオ提出ガイド

> AI活用能力を確認するためのポートフォリオです。  
> 以下の7つの項目から**2つ以上**を含めて作成をお願いします。

## ⚠️ 必須遵守事項

### 🔒 セキュリティ遵守
- 前職の**機密データ**や**ソースコード**が含まれないようご注意ください
- **公開データセット**または**モックアップデータ**を活用してください

### 📊 貢献度の明示
- プロジェクト進行時の役割と生成AIツールの活用範囲を具体的に明示してください
- 例：`アーキテクチャ設計 100%、コード実装 50% + AI補助 50%`

### 🔍 問題解決過程
- 単純な成果物よりも**データ前処理**、**モデル選定理由**、**トラブルシューティング過程**が含まれた文書を推奨します

### ▶️ 実行可能性
- コードを確認できる**GitHubリンク**または**デモURL**を添付してください



## 📋 含める項目（2つ以上選択）

### 1. AIを活用して解決した問題または自動化事例

**含める内容：**
- どのような状況/問題であったか
- AIをどのように活用したか
- 結果がどのように改善されたか
- 前後比較または改善数値

**作成例：**
```markdown
## 問題状況
- 毎週100件以上の顧客問い合わせを手動で分類するのに平均2時間かかっていた

## AI活用方法
- Claude APIを活用した自動分類システムを構築
- プロンプトエンジニアリングでカテゴリ分類精度を向上

## 改善結果
- 処理時間：2時間 → 10分（92%短縮）
- 分類精度：95%
```
